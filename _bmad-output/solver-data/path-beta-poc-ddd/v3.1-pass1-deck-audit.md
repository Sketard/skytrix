## Path skeleton (hypothesis)

The PvP combo path likely follows this structural skeleton:

1. **Set Pendulum scales** via search engine (need ≥1 scale-search). Most likely: tutor + place Kepler (scale 10) + a scale-1 (Gryphon/Surveyor/Copernicus) in P-zones. But what triggers a tutor first? The deck has:
   - **Nightmare Throne** field spell: on activation → take 1 Fiend with 0 ATK/DEF. **Doom Queen has 3000/0** → **DEF 0** ✓. Take Doom Queen → **destroy** it. Doom Queen destroyed in deck, sent to ED face-up (Pendulum monsters destroyed in deck go to ED face-up). This triggers Doom Queen's "if D/D/D or Dark Contract destroyed while Doom Queen in face-up ED…" — **but** Doom Queen IS the destroyed card here (it was sent **from deck**, not "you control"). Actually: the destruction happens **before** Doom Queen reaches the ED. The destroy occurs while it's still in the deck. So Doom Queen Machinex never reaches ED face-up via this route — it's already destroyed and sent to ED face-up (destroyed Pendulum → ED face-up). That puts Doom Queen face-up in ED but no other "D/D/D or Dark Contract" got destroyed concurrently. So no auto-SS.
2. Set Doom Queen as Pendulum Spell → activate it → triggers its Pendulum effect ability → place a Dark Contract from deck face-up (e.g., Zero King).
3. Use Zero King's effect: target a D/D you control (Doom Queen as scale!) → destroy it → SS a D/D from deck (e.g., Kepler). Destroying Doom Queen in P-zone sends it to ED face-up. This is the **key trigger**: now Zero King is on field, a "Dark Contract destroyed" event happens? **No** — Zero King wasn't destroyed; Doom Queen was.
4. Wait — Zero King's text: "destroy 1 D/D card you control (except itself); SS 1 D/D from deck, also you cannot SS for the rest of this turn except D/D".
5. The **face-up-in-ED self-SS** of Doom Queen Machinex requires destroying a *different* D/D/D OR Dark Contract while Doom Queen is in face-up ED. So:
   - First, Doom Queen needs to be in face-up ED.
   - Then, destroy a Dark Contract (e.g., Zero King).
   - Triggers: SS Doom Queen back to MZONE.
6. This is the recursive engine: place scales → activate Pendulum scale → destroy → SS Doom Queen → continue building.
